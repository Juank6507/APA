# Suma de Dos Números

Objetivo: Crear una API que reciba dos números enteros y retorne su suma en formato JSON.

Inputs:
- a: entero
- b: entero

Output esperado:
- JSON con el resultado de la suma

Criterio de éxito:
- La API responde con un objeto JSON que contiene la clave "result" con el valor de la suma de a y b.

Archivos:
- main.py: Archivo principal con la implementación de la API.
