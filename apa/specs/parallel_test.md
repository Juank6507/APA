# Objetivo: Generar tres funciones matemáticas independientes

Genera tres funciones simples, cada una en su propio archivo Python, que realicen operaciones matemáticas básicas. Las funciones son independientes entre sí (no se llaman unas a otras).

## Tarea 1: Función suma
- Archivo: `suma.py`
- Función: `sumar(a, b)` que retorna `a + b`
- Prueba: `assert sumar(2, 3) == 5`

## Tarea 2: Función resta
- Archivo: `resta.py`
- Función: `restar(a, b)` que retorna `a - b`
- Prueba: `assert restar(5, 3) == 2`

## Tarea 3: Función multiplicación
- Archivo: `multiplicacion.py`
- Función: `multiplicar(a, b)` que retorna `a * b`
- Prueba: `assert multiplicar(4, 3) == 12`
