# Herramienta CLI de transformación de texto

Objetivo: Implementar una herramienta de línea de comandos que acepte --input y --output, lea el archivo de entrada, convierta todo el texto a mayúsculas y lo guarde en el archivo de salida.

Inputs:
- --input: ruta del archivo de texto a leer
- --output: ruta del archivo de salida

Output esperado: Archivo de salida con el contenido del input en mayúsculas.

Criterio de éxito: Probar con un archivo de entrada de ejemplo y verificar que el archivo de salida contiene exactamente el texto transformado.
